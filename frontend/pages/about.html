About EventsHub

EventsHub is a lightweight front-end platform prototype for discovering, creating, and registering for events. It demonstrates client-side user management, event browsing, and registration workflows using LocalStorage and SessionStorage.

Goals: simplify event discovery, enable fast prototyping for admin CRUD, and provide a clean responsive UI suitable for extension with a backend later.

Tech: HTML, CSS, Vanilla JavaScript. Future: add API layer, authentication tokens, admin moderation dashboard, and real-time seat tracking.

Features Snapshot

  • User signup/login (client-side hashed placeholder)
  • Event listing with registration state
  • Dynamic navbar with role awareness (basic)
  • Responsive grid layout
